Transform your Mutant Ape Yacht Club (MAYC) NFT into a unique AI assistant with traits-based personality and specialized knowledge.

![Mutant Intelligence Screenshot](https://raw.githubusercontent.com/ianalloway/mayc-ai-assist-23/main/screenshot.png)

## Features

- **NFT Personalities** - Each MAYC has unique traits and AI personality based on their mutant characteristics
- **Specialized Skills** - Alpha, Mega, Toxic, Radioactive, and Prismatic mutants each excel in different areas
- **Visual Intelligence** - Your MAYC AI can analyze, understand, and work with visual content and data
- **Personal Assistant** - Each NFT becomes a dedicated AI agent tailored to your specific needs
- **Trait-Based AI** - Laser Eyes for precision, Gold Teeth for wealth insights, Cyber Helmet for tech expertise
- **Mutant Memory** - Your MAYC remembers everything and learns from your interactions over time
- **Wallet Connection** - Connect your wallet to verify MAYC ownership and unlock your AI assistant

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS
- **UI Components:** shadcn/ui
- **Web3:** wagmi, RainbowKit, Ethereum
- **Build Tool:** Vite
- **Styling:** Custom dark theme with vibrant gradients

## Getting Started

```bash
# Clone the repository
git clone https://github.com/ianalloway/mayc-ai-assist-23.git

# Navigate to project directory
cd mayc-ai-assist-23

# Install dependencies
npm install

# Start development server
npm run dev
```

## Deployment

The site is deployed on Netlify with automatic deployments from the main branch.
